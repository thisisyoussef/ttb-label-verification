# Eval Label Corpus

Store or reference the core label scenarios for this project here.

## Required scenarios

Use the manifest template as the canonical list of the six baseline cases:

- perfect spirit label
- warning text defect
- cosmetic brand mismatch
- wine dependency failure
- forbidden beer ABV format
- low-quality image

## Asset guidance

- Prefer stable filenames and keep them aligned with manifest IDs.
- If the real assets are stored elsewhere temporarily, keep the intended paths and notes in the manifest until the files are added.
- Do not silently swap a scenario without updating the manifest and relevant eval records.
