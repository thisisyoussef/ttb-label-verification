# Continue / Next Story

1. Read `docs/process/SINGLE_SOURCE_OF_TRUTH.md`.
2. If the current story is still in progress for your lane, continue it.
3. If the current branch already contains finished but unpublished work, finish the git flow first.
4. Otherwise take the next story SSOT points to for your lane.
5. Before editing, confirm the current checkout matches that story.
6. If it does not, switch to the correct story branch. Use or reopen a worktree only if that task was isolated that way.
